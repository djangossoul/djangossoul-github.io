Hargo

Hargo est un thème de boutique en ligne moderne, épuré et frais pour Hugo. Ce thème e-commerce est parfait pour les produits naturels, mais peut également être utilisé avec d’autres types de produits. Vous pouvez mettre à jour ce modèle pour qu’il corresponde à votre marque ou à votre entreprise et améliorer votre présence en ligne. Sa rapidité vous permettra de vous démarquer de la concurrence.

Fonctionnalités - Google analytics support
- CSS and Js bundle with hugo pipe
- Netlify settings predefine
- Contact form Support
- Compitable with snipcart
- Google map support
- Google page speed optimized ( 64% )
- Open graph meta tag
- Twitter card meta tag
Librairies - Bootstrap
- Jquery
- Themify Icons
- Slick Slider
- Google Fonts
