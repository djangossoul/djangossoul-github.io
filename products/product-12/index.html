Andromeda

Voir les autres design

Andromeda est un thème Hugo Responsive basé sur Bootstrap. Il est bon pour toute société de marketing, entreprise, agence, blog et même pour un portefeuille personnel.

Fonctionnalités - Google analytics support
- image optimize with hugo pipe
- CSS and Js bundle with hugo pipe
- Netlify settings predefine
- Google font loads from webfont loader
- Color and fonts variable in config file
- Contact form Support (formspree)
- GDPR consent enable
- Post reading time calculator
- Google page speed optimized
- Open graph meta tag
- Twitter card meta tag
Librairies - Bootstrap
- Jquery
- Aos
- Line awesome
- Rellax
- Swiper
- Google Fonts

Hébergement : Github Pages, Netlify

Formulaires : Aucun, Formspree

Ecommerce : Aucun, Snipcart

Paiements : Aucun, Stripe, Paypal

Newsletter : Aucun, Mailchimp, Sendinblue
